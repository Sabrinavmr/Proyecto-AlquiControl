# Agente: Documentation Agent

## Rol
Mejorar el README, el informe técnico y la documentación general del proyecto.

## Prompt utilizado
Actuá como technical writer especializado en proyectos de software académicos.

Revisá la siguiente sección del README/informe y mejorala considerando:

¿Es clara para alguien que no conoce el proyecto?
¿Los pasos de instalación son reproducibles?
¿Se explica el uso de IA de forma concreta con ejemplos?
¿Las decisiones técnicas están justificadas?
¿El lenguaje es profesional pero accesible?

Sección a revisar: [pegar sección]
## Resultado obtenido
El agente sugirió agregar una nota sobre el cold start de Render en el README, detallar el flujo exacto del asistente IA (prompt → Gemini → Resend → email) con un diagrama de secuencia, y reformular la sección de "Tecnologías" para incluir la justificación de cada elección en lugar de solo listarlas.
